HTML TAGS & THEIR DETAILS

  1. !DOCTYPE: Defines the document type and version of HTML used in the document.

  2. html: The root element of an HTML document, which contains all other elements.

  3. head: Contains metadata about the document, such as the title, charset, and links to external stylesheets or scripts.

  4. title: Sets the title of the page, which appears in the browser's title bar and in search engine results.

  5. body: Contains the content of the HTML document.

  6. h1-6: Heading elements, which define headings of varying levels of importance (h1 being the most important).

  7. p: A paragraph of text.

  8. span: A generic inline element, often used to apply styles or semantics to a portion of text.

  9. div: A generic block-level element, often used to group elements or apply styles to a section of content.

  10. a: A hyperlink, which links to another webpage or email address.

  11. img: An image element, which embeds an image into the page.

  12. ul,ol,dl: Unordered, ordered, and definition lists, respectively.

  13. li: A list item, used within lists.

  14. table: A table element, which defines a table with rows and columns.

  15. tr: A table row element, which defines a row within a table.

  16. td: A table data element, which defines a cell within a table.

  17. button: A button element, which defines a clickable button.

  18. style: A style element, which defines CSS styles for the page.

  19. link: A link element, which links to an external stylesheet or other resource.

  20. meta: A meta element, which provides metadata about the document.
